## 🚀 HOW TO USE IN YOUR CODE

```dart
// Import in any screen/widget
import 'package:flutter_gen/gen_l10n/app_localizations.dart';

// Use anywhere you have BuildContext
Text(AppLocalizations.of(context)!.playNow)
AppBar(title: Text(AppLocalizations.of(context)!.settings))

// With parameters
Text(AppLocalizations.of(context)!.nCards(10))  // "10 Cards"
Text(AppLocalizations.of(context)!.deckDeleted('Animals'))  // "Animals deleted"

// For decks
final locale = Localizations.localeOf(context).languageCode;
final name = deck.getLocalizedName(locale);
final description = deck.getLocalizedDescription(locale);
```
